# roms4me

A cross-platform ROM organizer and verifier for Linux, Windows, and macOS.

roms4me helps you manage your ROM collections by verifying them against DAT files — XML-based databases that describe expected ROM sets with checksums and metadata.

## Features

- **DAT file support** — Import and manage DAT files (Logiqx XML format)
- **ROM scanning** — Verify ROM ZIPs against DAT databases with CRC32 checksum validation
- **Status overview** — See at a glance which ROMs are OK, missing, or have bad dumps
- **Web UI** — Responsive browser-based interface that works on desktop and mobile
- **Cross-platform** — Runs on Linux, Windows, and macOS
